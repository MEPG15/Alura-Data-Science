# 📊 Alura Store - Análisis de Ventas y Rendimiento

Este proyecto forma parte de un desafío de Data Science para ayudar a la toma de decisiones estratégicas. El objetivo principal es evaluar el desempeño de cuatro tiendas para decidir cuál de ellas debería venderse para reinvertir en un nuevo negocio.

## 📝 Descripción del Proyecto
El señor Juan necesita identificar la tienda con menor rendimiento basándose en datos reales de ventas, satisfacción del cliente y costos logísticos.

## 🛠️ Herramientas Utilizadas
* **Python 3.10**
* **Pandas** (Manipulación de datos)
* **Matplotlib / Seaborn** (Visualización)
* **Google Colab** (Entorno de desarrollo)

## 📈 Análisis Realizados (Reto 1)
En esta primera etapa, evaluamos cinco aspectos clave:
1. **Facturación Total:** Identificamos que la **Tienda 4** tiene los ingresos más bajos ($1,038,375,700).
2. **Categorías Populares:** Mapeo de los productos más vendidos por sucursal.
3. **Calificación Promedio:** La Tienda 3 lidera en satisfacción (4.05 ⭐), mientras que la Tienda 1 es la más baja (3.98 ⭐).
4. **Productos Estrella:** Identificación de los artículos con mayor y menor rotación.
5. **Costos de Envío:** La Tienda 4 es la más eficiente logísticamente ($23,459 promedio).

## 💡 Conclusión Preliminar
Aunque la Tienda 4 es eficiente en envíos, su baja facturación total (un 10% menos que la Tienda 1) la convierte en la principal candidata para la venta estratégica.
